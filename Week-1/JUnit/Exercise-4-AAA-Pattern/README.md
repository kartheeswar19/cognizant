# Exercise 4: Arrange-Act-Assert (AAA) Pattern

This example demonstrates the AAA pattern in JUnit 4.

- Arrange: Set up the test data and objects before running the behavior.
- Act: Execute the method or action being tested.
- Assert: Verify that the result matches the expected outcome.
- @Before: Runs before each test to initialize shared test state.
- @After: Runs after each test to clean up resources and reset state.
